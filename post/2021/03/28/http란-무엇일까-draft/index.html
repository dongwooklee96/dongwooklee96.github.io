HTTP

  • HTTP(Hyper Text Transfer Protocol)의 약자로서, 웹 상에서 정보를 주고 받을 수 있는 프로토콜이다.

  • HTTP는 요청(REQUEST), 응답(RESPONSE)을 통해서 정보를 주고 받으며, HTTP 메시지 구조는 크게 헤더(HEADER)와 바디(BODY)로 나누어진다.

응답 바디

  • HTML, CSS, JS 파일을 응답한 것을 확인할 수 있다.
  • 웹 서버가 응답하는 파일은 이것이 전부이고, 브라우저에는 응답받은 파일을 통해서 화면에 그려주는 것이다.
